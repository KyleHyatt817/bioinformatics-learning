## üîß Practice Program: Pythagorean Theorem Calculator

**File:** `Read_input.py`

This program calculates the square of the hypotenuse given two sides of a right triangle using the Pythagorean theorem (a¬≤ + b¬≤).

```python
# Read input
a, b = map(int, input().split())

# Calculate square of hypotenuse
hypotenuse_squared = a**2 + b**2

# Output
print(hypotenuse_squared)
```

**What it demonstrates:**
- Reading multiple inputs on one line
- Type conversion with `map(int, ...)`
- Mathematical operations with `**` (exponentiation)
- Clean code structure with comments
