### Type Conversion Deep Dive
```python
# Integer conversion
x = int("42")           # String to int
y = int(3.14)           # Float to int (truncates)

# Float conversion
a = float("3.14")       # String to float
b = float(5)            # Int to float

# String conversion
s = str(42)             # Int to string
t = str(3.14)           # Float to string
```

### Mathematical Operations
```python
# Basic arithmetic
addition = 5 + 3        # 8
subtraction = 10 - 4    # 6
multiplication = 6 * 7  # 42
division = 15 / 4       # 3.75 (float)
floor_division = 15 // 4 # 3 (int)
modulo = 15 % 4         # 3 (remainder)
exponent = 2 ** 3       # 8 (2 to the power of 3)
```

### String Formatting
```python
# Modern f-strings (Python 3.6+)
name = "Alice"
age = 25
print(f"{name} is {age} years old")

# Format method
print("{} is {} years old".format(name, age))

# String concatenation
greeting = "Hello, " + name + "!"
```
